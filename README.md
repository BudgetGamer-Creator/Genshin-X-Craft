# Genshin X Craft

**Author:** BudgetGamer1503  
**Version:** 1

A Minecraft Bedrock Add-on that brings the world of Teyvat to Minecraft.

## Features

### Entities
* **Paimon:** Your travel companion (includes custom sounds and animations).
* **Elemental Slimes:** Anemo, Cryo, Dendro, Electro, Geo, Hydro, and Pyro variants.
* **Hilichurls:** Basic enemies found in the world.

### Weapons & Items
* **Traveler's Sword**
* **Waster Greatsword**
* **Dull Blade**
* **Hunter's Bow**
* **Beginner's Protector**
* **Apprentice's Notes**

## Installation
1. Download the latest `.mcaddon` from the Releases tab.
2. Double-click to import into Minecraft Bedrock.

3. Activate the Behavior Pack and Resource Pack in your world settings.
